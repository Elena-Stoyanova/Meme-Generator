### Run Locally

Clone the project

```bash
  git clone https://github.com/Elena-Stoyanova/Meme-Generator.git
```

Go to the project directory

```bash
  cd Meme-Generator
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
